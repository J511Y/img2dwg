# web_image2cad_v1

Compliant web-source benchmark seed set for img2dwg.

## Scope
- Includes both **JPG** and **PNG** samples.
- Each image row includes a paired **DXF candidate URL**.
- All rows include SPDX license + source repository + pinned source commit.

## Files
- `manifest.csv`: dataset manifest (URL, SHA-256, license, attribution metadata)
- `ATTRIBUTION.md`: human-readable attribution summary

## Reproducibility
SHA-256 values in `manifest.csv` were verified against direct downloads from pinned raw GitHub commit URLs.
