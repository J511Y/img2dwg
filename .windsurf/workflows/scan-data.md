# Scan Data Workflow

데이터 폴더를 스캔하고 이미지-DWG 쌍을 분류하는 워크플로우입니다.

## 작업 순서

1. **데이터 폴더 확인**
   - `datas/` 폴더 구조 파악
   - 상위 폴더 목록 출력 (예: 2501 (2), 2501 (3) 등)

2. **프로젝트 폴더 스캔**
   - 각 상위 폴더 내의 프로젝트 폴더 확인
   - 프로젝트별 파일 목록 수집

3. **파일 분류**
   - **변경 관련 파일**:
     - 파일명에 "변경"이 포함된 이미지 파일 (jpg, png)
     - 파일명에 "변경"이 포함된 DWG 파일
   - **단면도 관련 파일**:
     - 파일명에 "단면"이 포함된 이미지 파일
     - 파일명에 "단면"이 포함된 DWG 파일

4. **통계 출력**
   - 전체 프로젝트 수
   - 변경 관련 쌍의 개수
   - 단면도 관련 쌍의 개수
   - 불완전한 쌍 목록 (이미지만 있거나 DWG만 있는 경우)

5. **결과 저장**
   - `output/scan_results.json`에 분류 결과 저장
   - CSV 형식으로도 저장 (`output/scan_results.csv`)

## 실행 명령어

```bash
python scripts/scan_data.py
```

## 예상 출력 형식

```
=== 데이터 스캔 결과 ===
총 프로젝트 수: 150
변경 관련 쌍: 120
단면도 관련 쌍: 145

불완전한 쌍:
- 이매촌 진흥 814-405: 변경 이미지만 존재
- 안성 공도 101-904: DWG 파일 누락
```
